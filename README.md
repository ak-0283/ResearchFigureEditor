## 📖 About The Project

Researchers, students, and developers often create architecture diagrams, flowcharts, system designs, and research figures using tools like **Draw.io, Canva, Figma, Excalidraw, Lucidchart, and PowerPoint**.

After exporting the diagram, they frequently face challenges such as:

❌ Incorrect dimensions for publication

❌ Extra whitespace around figures

❌ Poor background colors

❌ Wrong aspect ratios

❌ Need to reopen Draw.io or Canva for small modifications

❌ Time-consuming manual adjustments

❌ Multiple tools required for simple edits

### 💡 Solution

**Research Figure Studio** eliminates these frustrations by providing a dedicated platform where users can upload their existing diagrams and instantly optimize them for academic publishing.

No need to reopen Canva, Draw.io, Figma, or any other design software.

Simply:

📤 Upload → ✨ Edit → 📥 Export

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/research-figure-studio.git
```

### Navigate to Project

```bash
cd research-figure-studio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
